# MerryWeather App
This is an iOS Project developed to forecast weather Hourly(24hrs) or Daily(8 Days Forecast). Makes use of Alamofire utility and Openweather API to fetch data. 
## Versions Used
1. Xcode = 13
2. iOS = 15.2
3. CocoaPods = 1.11.2
